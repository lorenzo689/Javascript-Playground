# JavaScript Playground

A collection of small projects built with pure HTML, CSS, and JavaScript to practice core web development concepts without frameworks.

## Focus
- DOM manipulation
- Event handling
- Basic application logic
- UI interactions and state
- Clean separation of structure, style, and behavior

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

## Purpose
This repository is used as a learning playground to strengthen fundamentals and better understand how modern web interfaces work under the hood.

## Run locally
Simply open the `index.html` file in your browser  
or use a local server (e.g. VS Code Live Server).

## Notes
No frameworks or libraries are used — the goal is to build everything from scratch and focus on fundamentals.
